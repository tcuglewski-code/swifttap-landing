# Zipayo Landing Page

Landing page für [Zipayo](https://zipayo.vercel.app) — Moderne Zahlungslösungen für KMU.

## Features

- QR-Code Zahlungen
- Tap to Pay (NFC)
- Terminal-Integration
- Plattform-Integration (API)

## Tech Stack

- Next.js 14
- Tailwind CSS
- Material Design Icons
- Vercel Hosting

## Development

```bash
npm install
npm run dev
```

## Deployment

Automatisch via Vercel bei Push auf `main`.
